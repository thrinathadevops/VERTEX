
# VAREX Frontend (Next.js 14)

Production-ready SaaS UI for the VAREX platform using Next.js 14 (App Router), TypeScript, Tailwind CSS, and integration with the FastAPI backend.

## Getting started

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

Backend is expected at `NEXT_PUBLIC_API_BASE_URL` (e.g. `http://localhost:8000`).
